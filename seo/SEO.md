### title
##### 强调重点，重要的关键词不能超过2次，不同页面的title要有所不同
```
<title></title>
```
### description
##### 对页面内容高度概括，不可堆砌关键词，不同页面的description要有所不同
```
<meta name="description" content="">
```
### keyworlds
##### 列举出关键词即可
```
<meta name="description" content="">
```
### HTML语义化标签
```
<header></header>
<footer></footer>
```
